### CREE8 KickStart: OnBoard Enterprise Client Accounts

This repository hosts IaaC for the Enterprise Clients to onboard their accounts to the CREE8 platform.

#### Tech Stack

- AWS CloudFormation
- IAM (Identity and Access Management)
